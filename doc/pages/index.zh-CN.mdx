## 概览 [#overview]

```ts
import { Earth } from "dde-earth";
import { TIFFLayerLoader } from "@dde-earth/plugin-tiff-loader";

const earth = new Earth("container");

// 使用插件，加载tiff
earth.usePlugin(new TIFFLayerLoader());

earth.addLayer({
  url: "./tiff.tiff",
  method: "tiff",
  renderOptions: {
    single: {
      colorScale: "rainbow",
    },
  },
});
```

该示例中，在一行代码初始化地球完成后，通过添加`TIFFLayerLoader`这一插件，使`earth`拥有了加载`tiff`的能力。

完备的TypeScript机制，使得**dde-earth**拥有智能的代码提示。开发者此时会发现调用`addLayer`方法时，多了一种`tiff`类型的图层。

## 特性 [#features]

仅需一行代码，你就可以省下项目中对cesium的大量封装工作，并立即拥有以下这些不可思议的特性：

- 轻量化（核心包gzip压缩后仅**3.9kb**）
- 支持通过**插件**快速扩展earth能力
- 支持 **TypeScript**
- 支持**多语言**
- **图层**管理
- **地形**管理
- 内置事件订阅（图层添加、图层移除、图层渲染、多语言切换、地形切换…）
- 图层操作（缩放、渲染）
- ……

此外，官方提供了丰富的插件可供选择：
- 鼠标事件订阅
- 图层顺序交换
- 基础图层加载（tms、wmts、wms、arcgis）
- **GeoJson**加载渲染
- **Mapbox矢量瓦片**加载渲染
- **NetCDF**加载渲染
- **TIFF**加载渲染
- 导航工具
- ……

以及 [更多](/docs/getting-started)。
